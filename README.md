# NYC Taxi Spark ETL Pipeline

Distributed ETL pipeline for processing NYC Taxi Trip data using PySpark.

## Project Overview

This project demonstrates a scalable Spark ETL workflow for large-scale taxi trip analytics.

The pipeline includes:
- data ingestion
- cleaning and validation
- Spark transformations
- partitioned parquet output
- modular ETL structure
- distributed processing concepts

## Tech Stack

- PySpark
- Spark SQL
- Python
- Parquet
- Jupyter Notebook

## Features

- Large-scale data processing
- Modular ETL architecture
- Data cleaning and transformation
- Optimized parquet outputs
- Spark-based analytics
